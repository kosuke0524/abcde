# Life Path Board

人生ゲーム風のすごろくアプリです。ブラウザだけで動きます。

## 遊び方

`index.html` をブラウザで開くと起動します。

- 1人から4人までの人数を選んで遊べる
- 各プレイヤーが仕事、大学、起業、イベント、家族、投資からコースを選ぶ
- コースごとに収入、リスク、幸せ、イベント成功率の補正が変わる
- `Spin` でルーレットを回して進む
- 止まったマスのイベントや選択を解決する
- 70マスの盤面で、買い物、転職、教育、移住、相続、終盤イベントなどの分岐が出る
- 選択マスでは複数の選択肢カードから行動を選び、結果がストーリーとして表示される
- 一部の選択マスではイベント後に進むルートを選べる
- 収入、出費、家族、選択、ゴールでは種類別の演出が出る
- 職業は仕事コースと大学コースで複数候補から決まる
- ゴール時に現金、借金、幸せ、家族、スキルから最終スコアを計算

キーボードでは `Space` でもルーレットを回せます。
